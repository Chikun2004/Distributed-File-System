# Frontend Styling and Theming Guide

## Design System

### Colors
```css
/* Primary Colors */
--primary-main: #1976d2;
--primary-light: #42a5f5;
--primary-dark: #1565c0;

/* Secondary Colors */
--secondary-main: #9c27b0;
--secondary-light: #ba68c8;
--secondary-dark: #7b1fa2;

/* System Colors */
--error: #d32f2f;
--warning: #ed6c02;
--info: #0288d1;
--success: #2e7d32;
```

### Typography
```css
/* Font Families */
--font-primary: 'Roboto', sans-serif;
--font-secondary: 'Open Sans', sans-serif;

/* Font Sizes */
--text-xs: 0.75rem;
--text-sm: 0.875rem;
--text-base: 1rem;
--text-lg: 1.125rem;
--text-xl: 1.25rem;
```

### Spacing
```css
/* Spacing Scale */
--spacing-xs: 0.25rem;
--spacing-sm: 0.5rem;
--spacing-md: 1rem;
--spacing-lg: 1.5rem;
--spacing-xl: 2rem;
```
